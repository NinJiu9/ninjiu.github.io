节目的第一站来到了新民学会旧址，以“远方s的星火”为主题，着重讲述百多年前，蔡和森在毕业谋职时逐渐获得有关旧中国社会问题的深层认识的思想成熟过程，以及参与组建新民学会和为探求救国真理而赴法勤工俭学的故事。 第二站来到了新疆兵团军sd垦博物馆、军垦第一连景区等地，以“远方的胡杨”为主题，讲述了80多年前，援疆湘女万里赴疆、艰苦创业的故事，并深入展现湘女代表当年应征入伍时的抉择心态与青春志向。 在一处处案例实景中穿行，结合授课环境中的实景s与实物展开讲解，让参与者充分感受“沉浸式实景授课，走读式移动讲述”的独特魅力。同时还设置了时空对接、创意短视频等环节，并采用情景漫画等手段大幅度地丰富了画面信息，使得节目内容更加生动。也让我有所收获。
